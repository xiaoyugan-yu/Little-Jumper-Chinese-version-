# 小跳跃者（浏览器平台跳跃小作）- 中文版

说明：
- 这是一个完整的浏览器平台跳跃小游戏（单页面）。包含：多关、金币、敌人、尖刺、旗帜出口、触控支持、音效、最高分本地存储与中文界面。
- 运行方法：将 `index.html` 打开在现代浏览器（推荐 Chrome / Edge / Safari）即可开始游戏。

操作：
- 键盘：← / → 或 A / D 移动，↑ 或 空格 / W 跳跃。
- 移动端：屏幕会显示左/右/跳跃按钮，支持触控。
- 在主菜单可以切换音效与选择关卡。

文件：
- index.html — 游戏全部代码（包含样式与脚本），直接打开即可运行。
- 本游戏美术为矢量 SVG 嵌入，运行时无需外部资源。

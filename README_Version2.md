# 小跳跃者 — 升级版 （浏览器平台跳跃游   ）

说明
- 这是小跳跃者的升级版本：更多关卡（6 关示例）、角色多帧动画、背景音乐（可开/关）、更多道具（加速 B / 双跳 D / 护盾 H / 宝石 G）、移动平台、粒子特效与物理改进（coyote time、jump buffer）。
- 依然为单文件实现：打开 `index.html` 即可在现代浏览器运行（建议 Chrome / Edge / Safari）。

新特性概览
- 关卡：增加到 6 个示例关卡，包含移动平台与更多道具分布。
- 角色动画：使用多帧 SVG 切换实现 idle / run / jump 帧。
- 背景音乐：基于 WebAudio 的简单循环序列（在用户交互后开始），支持开关。
- 道具：
  - B（加速）：短时间提高移动加速度与速度上限；
  - D（双跳补充）：补回双跳能力；
  - H（护盾）：抵消一次伤害；
  - G（宝石）：高分收集物；
- 物理改进：加入 jump buffer（预按跳跃容忍）与 coyote time（离地后短时间仍可跳），让平台跳跃手感更好。
- 粒子特效：金币/宝石/死亡/跳跃会触发简单粒子效果。

运行与操作
- 打开 index.html 即可开始。
- 键盘：← / → 或 A / D 移动；↑ 或 空格 / W 跳跃。
- 移动端：会显示触控按钮（左/右/跳跃）。
- 菜单中可以切换音效与 BGM（音乐在首次交互后自动启用以符合浏览器策略）。
